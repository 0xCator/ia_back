## Login

### ``api/User/Login`` - POST
#### Request
```json
{
	"username": "
	"password": ""
}
```
#### Response
The response contains the user's information (and an auth token once implemented).

### ``api/User/Register`` - POST
#### Request
```json
{
	"username": "",
	"password": "",
	"email": "",
	"name": "",
}
```
#### Response
The response contains the user's information (and an auth token once implemented).
